            • 封一个IP的命令是：
iptables -I INPUT -s 1.1.1.1 -j DROP
解封一个IP的命令是：
iptables -D INPUT -s 1.1.1.1 -j DROP

封几个段的命令是：

封IP段的命令是：
iptables -I INPUT -s 1.1.0.0/16 -j DROP
iptables -I INPUT -s 1.2.0.0/16 -j DROP
iptables -I INPUT -s 1.3.0.0/16 -j DROP
封整个段的命令是：
iptables -I INPUT -s 1.0.0.0/8 -j DROP
iptables -I INPUT -s 2.0.0.0/8 -j DROP
iptables -I INPUT -s 3.0.0.0/8 -j DROP
